# 열흘의 마을 (Village of Ten Days)

RPG Maker 스타일의 10일 루프 마을 생활 시뮬레이션 웹 게임

## 게임 소개

도시에서 지친 직장인이 산속 작은 마을 "달빛 마을"에서 10일간의 휴가를 보내는 이야기입니다.
전투 없이 NPC와의 대화와 선택을 통해 스토리가 진행되며, 누적된 선택에 따라 4개의 엔딩으로 분기합니다.

## 실행 방법

```bash
# 로컬 서버로 실행 (ES Modules 사용으로 서버 필요)
npx http-server . -p 8080 -c-1
# 브라우저에서 http://localhost:8080 접속
```

또는 VS Code의 Live Server 확장으로 `index.html`을 열어도 됩니다.

## 게임 구조

- **10일 루프**: 매일 2~3번의 행동으로 마을 장소를 방문하고 NPC와 대화
- **5명의 NPC**: 각각 고유한 스토리라인과 호감도 시스템
- **핵심 플롯**: 달빛 축제 준비 + 20년 전 화재의 비밀
- **4개의 엔딩**: 진엔딩, 연희 엔딩, 비밀 엔딩, 배드엔딩

## NPC 소개

| NPC | 나이 | 장소 | 특징 |
|-----|------|------|------|
| 연희 | 25 | 마을 회관 | 이장의 딸, 마을 살리기에 헌신 |
| 달수 | 40 | 장터 | 상인, 장군과 10년 갈등 |
| 해미 | 30 | 개울가 | 화가, 마을 비밀을 알고 있음 |
| 장군 | 60 | 당산나무 | 관리인, 과거의 죄책감 |
| 보미 | 17 | 민박집 | 마을 유일한 청소년, 마을 역사 기록 |

## 기술 스택

- 순수 HTML/CSS/JavaScript (ES Modules)
- 프레임워크/라이브러리 없음
- localStorage 세이브/로드
- CSS 기반 레트로 RPG UI

## 프로젝트 구조

```
NLP06/
├── index.html          # 진입점
├── css/                # 스타일시트
├── js/
│   ├── main.js         # 게임 컨트롤러
│   ├── engine/         # 상태관리, 이벤트엔진, 렌더러
│   ├── screens/        # 타이틀, 마을맵, 대화, 요약, 엔딩
│   └── utils/          # 타자기 효과, 오디오
└── data/
    ├── config.js       # 설정값
    ├── npcs.js         # NPC 정의
    ├── locations.js    # 장소 정의
    ├── events.js       # 일별 이벤트
    ├── endings.js      # 엔딩 조건/텍스트
    └── dialogue/       # NPC별 대화 트리
```
